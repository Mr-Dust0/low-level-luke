# Agents Guide

Agent-facing instructions mirroring `CLAUDE.md` files in this repo.

## Project

- Project type: Hugo static site (PaperMod theme).
- Repo root: `/home/john/blog`.

## Global Workflow

After content or config changes:

1. Run `hugo` from repo root.
2. Commit changed files (including `public/` updates when generated).
3. Push.

## Braindumps Workflow

For files in `/content/braindumps/`:

1. Keep user's raw notes exactly as written in `## My Notes`.
2. Add `## What I Got Wrong` with specific corrections and missing concepts.
3. Add `## How [Topic] Actually Works` with a clean explanation.

Required frontmatter fields:

- `title`
- `date`
- `tags` (must include `"braindump"` first)
- `summary`
- `draft`

Required page flow:

1. Frontmatter
2. Intro italic active-recall line
3. `## My Notes`
4. `## What I Got Wrong`
5. `## How [Topic] Actually Works`
6. `### Tradeoffs` (when relevant)
7. Closing P.S. line

Correction style:

- Be specific about what was wrong.
- Explain why the distinction matters.
- Include important omissions.
- Keep tone direct and non-condescending.
